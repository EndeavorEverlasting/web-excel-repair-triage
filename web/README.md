# Web Interfaces

Local web tools for the triage pipeline.

## Prompt Kit Control Panel

A self-contained dark-theme reference panel for the AI Harness Prompt Kit v39.

### Quick Start

```powershell
# From the repo root:
start web\prompt-kit\index.html
```

Or open `web/prompt-kit/index.html` directly in any browser.

### Features
- 58 prompts with one-click copy
- Search by name, type, ID, or keyword
- Synonym search ("generate artifact" finds P56)
- Color-coded section navigation with glowing dividers
- Mobile-friendly for phone use
- Keyboard shortcuts (`/` search, `1/2/3` tabs, `R` reference panel, `Esc` to clear)

### Hotkeys

| Key | Action |
|-----|--------|
| `/` | Focus search |
| `1` / `2` / `3` | All / Standard / GNHF |
| `R` | Toggle reference sidebar |
| `Esc` | Collapse > Clear > Unfilter > Reset |
